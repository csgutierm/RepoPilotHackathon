Project: RepoPilot

Goal:
Build a proof-of-concept platform that analyzes Python repositories and generates:

- architecture summary
- dependency graph
- suggested documentation
- test coverage recommendations
- onboarding report

Requirements:
- python 3.13.7
- FastAPI backend
- modular architecture
- AST-based analysis
- clean production code
- tests
- strong documentation

IBM Bob should:
- scaffold project
- explain generated architecture
- generate/refactor modules
- create tests
- document implementation decisions
